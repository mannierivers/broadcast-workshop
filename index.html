This report contains no data suitable for Chart.js/Plotly. -->

                            <h3 class="text-xl font-semibold mb-3 text-gray-800">Sony a7iii (The "Fast" Cam)</h3>

                            <ul class="list-decimal list-outside space-y-2 pl-5 text-gray-700">

                                <li>**Connection:** Camera HDMI Out → Video Capture Card → USB into PC.</li>

                                <li>**OBS Source:** Add a `Video Capture Device` source. Select your capture card from the dropdown.</li>

                                <li>**Audio:** DJI RX receiver plugged into Camera Mic port. Audio will travel via the HDMI cable.</li>

                                <li>**Camera Settings:** Ensure camera is in Movie mode and HDMI output is set to "Clean" (no on-screen info).</li>

                            </ul>

                        </div>

                        <div id="setup-panasonic" class="tab-content">

                            <h3 class="text-xl font-semibold mb-3 text-gray-800">Panasonic XH1600 (The "Network" Cam)</h3>

                             <ul class="list-decimal list-outside space-y-2 pl-5 text-gray-700">

                                <li>**Connection:** Connect camera via Ethernet cable to the same network router/switch as the OBS PC.</li>

                                <li>**Camera Settings:** Enable RTSP streaming in the camera's network menu and note its IP address.</li>

                                <li>**OBS Source:** Add a `Media Source`. Uncheck 'Local File'.</li>

                                <li>**Input:** Enter the camera's RTSP URL (e.g., `rtsp://192.168.1.xxx:554/stream`).</li>

                                <li>**Note:** This camera will likely have the most delay (latency). Be prepared to fix this in Phase 2.</li>

                            </ul>

                        </div>

                        <div id="setup-iphones" class="tab-content">

                            <h3 class="text-xl font-semibold mb-3 text-gray-800">iPhones (The "Cloud" Cams)</h3>

                             <ul class="list-decimal list-outside space-y-2 pl-5 text-gray-700">

                                <li>**Connection:** Connect phones to a reliable, high-speed WiFi network (5GHz preferred) or use 5G/LTE.</li>

                                <li>**VDO.Ninja Workflow (Stream Tech):**

                                    <ol class="list-alpha list-outside pl-5 mt-2 space-y-1">

                                        <li>Go to `vdo.ninja` on the OBS PC and click "Create Room".</li>

                                        <li>Send the invite link (or show the QR code) to the iPhone operators.</li>

                                        <li>iPhone operators join the room as a "Camera".</li>

                                        <li>Copy the "View link" for each specific camera from the VDO.Ninja director's panel.</li>

                                    </ol>

                                </li>

                                 <li>**OBS Source:** For each iPhone, add a `Browser Source`. Paste the unique VDO.Ninja "View link" into the URL field. Set width/height (e.g., 1920x1080).</li>

                            </ul>

                        </div>
